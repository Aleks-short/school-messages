# 🏫 Училищни Съобщения (School Messages)

Модерна платформа за комуникация и управление на образователни институции. Проектът е предназначен да улесни връзката между администрация, учители, ученици и родители чрез структурирана и сигурна система за съобщения.

---

## 🚀 Основни функции

*   **🔐 Сигурна Аутентикация:** Многослойна система за вход с роли (**Администратор**, **Директор**, **Учител**, **Ученик**).
*   **✉️ Усъвършенствана Система за Съобщения:** Поддръжка на форматиран текст (Rich Text), прикачени файлове и следене на статуса за прочитане.
*   **📊 Динамични Табла (Dashboards):** Персонализирана статистика и метрики в реално време за всяка роля.
*   **🏫 Управление на Училища и Класове:** Модули за организиране на образователната структура, паралелките и потребителите.
*   **🕵️ Одитен Лого (Audit Log):** Пълно проследяване на действията и промените в системата за максимална прозрачност и сигурност.
*   **🔔 Смарт Известия:** Вградена система за нотификации и автоматизирани напомняния.
*   **🌓 Модерен Интерфейс:** Напълно адаптивен (responsive) дизайн с поддръжка на Тъмен и Светъл режим (Dark/Light Mode).

---

## 🛠️ Технологичен стек

### **Frontend**
*   **React 18** с **TypeScript**
*   **Vite** - за ултра-бързо време за разработка и билд.
*   **Tailwind CSS** - за прецизно и модерно стилизиране.
*   **Shadcn/UI** & **Radix UI** - висококачествени, достъпни компоненти.
*   **Framer Motion** - плавни анимации и премиум потребителско изживяване.
*   **TanStack Query (v5)** - синхронизиране на данни и кеширане.
*   **React Router 6** - за интуитивна навигация.

### **Backend**
*   **Node.js & Express**
*   **SQL.js / SQLite** - надеждна база данни с лесна преносимост.
*   **BcryptJS** - криптиране на пароли.
*   **Multer** - обработка на файлови трансфери.
*   **TSX** - директно изпълнение на TypeScript в среда на Node.js.

## 🏁 Първи стъпки

### Предварителни условия
*   Уверете се, че имате инсталиран **Node.js** (препоръчително LTS версия).

### Инсталация

1. Инсталирайте всички необходими пакети:
   ```bash
   npm install
   ```

### Стартиране в режим на разработка

Проектът е конфигуриран да стартира едновременно фронтенда и бекенда с една лесна команда:

```bash
npm run dev
```

*   **Frontend (Vite):** [http://localhost:5173](http://localhost:5173)
*   **Backend API:** [http://localhost:3001/api](http://localhost:3001/api)
